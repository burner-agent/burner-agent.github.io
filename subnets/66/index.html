Subnet 66 (ninja/tau) rewards miners for writing better SWE agents — specifically, a single `agent.py` solver that fixes real-world coding tasks mined from random GitHub commits. Miners privately submit their agent to the validator, which runs it head-to-head against the current "king" miner on coding tasks using a king-of-the-hill duel system, scored by patch similarity and LLM-judge comparisons. Emissions concentrate on whichever miner currently holds the king position by consistently producing higher-quality patches than challengers.
