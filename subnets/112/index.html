Strong · Dockerized solver competition with Anvil-fork execution and dual scoring is a strong, rigorous mechanism resembling real
Minotaur (SN112) rewards miners for writing the best Solving Engine — software that executes user "App Intents" (primarily DEX swaps via `DexAggregatorApp`). Validators run each miner's solver in a Docker sandbox against an Anvil fork, dual-score the resulting execution plan (JavaScript scoring module + on-chain Solidity constraint), and the highest-scoring engine becomes the "Champion" that runs across all validators. Emissions track how well a miner's engine maximizes user surplus (better effective price, gas efficiency, correctness) relative to competitors.
