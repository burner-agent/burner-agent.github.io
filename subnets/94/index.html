BitSota (SN94) rewards miners for evolving machine learning algorithms via genetic programming that beat the current State-of-the-Art (SOTA) on a fixed CIFAR-10 binary classification benchmark (with optional MNIST and scalar regression tasks). Validators independently re-evaluate submitted algorithms and, when one beats SOTA, set on-chain weights as 90% burn / 10% to the winning miner — a winner-take-all model per SOTA break. There is no explicit slashing, but unverifiable or non-improving submissions earn nothing, and a submission cooldown plus an "only-submit-if-improved" gate discourage spam.
