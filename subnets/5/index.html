Hone miners solve ARC-AGI-2 abstract reasoning puzzles by submitting code that runs in a GPU sandbox. Validators measure exact-match accuracy on puzzle outputs and reward only the top 5 miners using exponential decay distribution, with 95% of emissions burned. Only the best-performing algorithm wins significant rewards.
