SOMA (netuid 114) rewards miners for submitting algorithms that solve MCP-server tasks — currently **context compression**, where miners' code must shrink input text while preserving the information needed to answer questions about it. Validators score submissions by running them in a sandbox and measuring compression ratio and downstream Q&A quality, with the weekly top-ranked miner taking the incentive allocation in a winner-take-most model.
