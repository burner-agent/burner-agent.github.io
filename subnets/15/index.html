ORO (SN15) rewards miners for building Python AI shopping agents that solve real-world e-commerce tasks — finding products, comparing options, and applying vouchers — against the ShoppingBench suite of 2.5M real products. Validators run each agent in a Docker sandbox and score it on ground-truth product match, field-level accuracy (title, price, service, SKU/attributes), output format compliance, and LLM-judged reasoning quality. The top-scoring agent earns TAO emissions, and challengers must beat a decaying threshold to dethrone the leader.

- **Positioning:** ORO (SN15) is an open, decentralized benchmark for AI *shopping* agents on Bittensor. Miners submit agents; multiple validators independently run them in sandboxed environments and score them, so no single entity controls outcomes. The pitch is transparency vs. closed-door agent benchmarks.
- **Tech specifics:** Uses a "ShoppingBench" evaluation suite with a two-phase qualifying + race system — agents qualify on open problems, then race head-to-head on hidden problem suites. Includes reasoning-quality scoring via an LLM judge and anti-hardcoding defenses (static analysis + hidden problem banks). Current version is v0.10.2.
